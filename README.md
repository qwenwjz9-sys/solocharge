# SoloCharge 🪫⚡

> 内向者的个人能量管理工具 — 安静陪伴，看见自己

一款专为 i 人（内向者）设计的「社交电量」追踪与管理轻量 App。帮助用户感知自身能量状态，通过记录耗电/充电事件，建立对自身能量节律的认知。

---

## 快速开始

### 方式一：直接打开（推荐）

访问 **[https://solocharge.app](https://solocharge.app)**（需自行部署，或直接用下方方式二）

### 方式二：本地运行

```bash
# 只需要一个 HTTP 服务器
cd prototype
python3 -m http.server 8800
# 打开 http://localhost:8800/index.html
```

### 方式三：部署到任意静态托管

```bash
# 将 prototype/ 目录部署到任意静态托管服务
# 支持：Vercel / Netlify / GitHub Pages / 阿里云OSS / 腾讯云COS
```

---

## 功能特性

| 功能 | 说明 |
|------|------|
| **电量显示** | 环形电量表，实时显示 0-100% 社交能量 |
| **快速记录** | 3 秒完成一次耗电/充电记录，低摩擦 |
| **趋势图表** | 今日/本周可视化分析能量变化 |
| **周报洞察** | 自动生成本周能量规律摘要 |
| **自定义事件** | 添加个人专属的耗电/充电事件 |
| **数据导出** | CSV 格式导出全部记录 |
| **离线可用** | PWA 支持，缓存后可离线使用 |
| **隐私优先** | 所有数据本地存储，不上传服务器 |

---

## 技术栈

- **纯前端**：单 HTML 文件，零依赖
- **数据存储**：localStorage（数据全在本地）
- **PWA 支持**：可添加到桌面，离线使用
- **设计语言**：深色模式优先，噪点纹理，线性图标

---

## 设计理念

- **安静优先**：默认不打扰，用户主动打开才有内容
- **一屏一事**：每个界面只传达一件事
- **无需学习**：首次打开无需引导，自然理解

## 部署到微信小程序（可选）

如需发布为微信小程序，可使用 WebView 嵌入此 HTML：

1. 将 `prototype/` 目录部署到 HTTPS 服务器
2. 在微信小程序中使用 `<web-view src="你的URL">` 加载
3. 配置小程序 `request` 域名白名单
